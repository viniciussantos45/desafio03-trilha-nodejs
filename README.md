# desafio03-trilha-nodejs

A **Repositories CRUD API** built as the third challenge for the [Rocketseat Ignite](https://www.rocketseat.com.br/ignite) Node.js bootcamp.

## Objective

The objective of this challenge is to build a REST API that manages code repository entries — storing title, URL, and tech stack — and implements a "like" system that increments a counter for each repository, validated with automated tests.

## Features

- List all repositories
- Create a repository (title, URL, techs)
- Update a repository (cannot update likes directly)
- Delete a repository
- Like a repository (increments counter)

## Tech Stack

- Node.js
- Express
- Jest

## Getting Started

```bash
yarn install
yarn dev

# Run tests
yarn test
```
